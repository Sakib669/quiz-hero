Q-1: What is the difference between find(), forEach(), map(), filter() methods ?

Ans: 1.1 find() method is used to loop through an array and it will return the first element which matches the condition.
1.2 forEach() method is used to loop through an array but it does not return anything.
1.3 map() is a shortcut method to loop through an array.
1.4 filter() method is used to loop through an array and it will return all the elements that match the condition.

Q-2: What is the difference among localStorage, sessionStorage and cookies ?

Ans: 2.1 Local storage - Stores data with no expiration date.
2.2 Session storage - Stores data for one session (data is lost when the browser tab is closed).
2.3 Cookies are text files with small pieces of data.

Q-3: What is Event Loop in JS ?

Ans: It is JavaScript's runtime model based even loop, It is responsible for handling and executing the code, collecting and processing events, and executing queued sub-tasks.
